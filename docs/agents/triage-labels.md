# Triage Labels

Skills 使用五种标准 triage 角色。此文件将角色映射到本仓库 issue tracker 中实际使用的标签名。

| 标准角色                  | 本仓库标签            | 含义                           |
| ------------------------- | --------------------- | ------------------------------ |
| `needs-triage`            | `needs-triage`        | 维护者需要评估此 issue         |
| `needs-info`              | `needs-info`          | 等待提问者补充更多信息         |
| `ready-for-agent`         | `ready-for-agent`     | 描述完整，可由 AFK agent 接手  |
| `ready-for-human`         | `ready-for-human`     | 需要人类开发者实现             |
| `wontfix`                 | `wontfix`             | 不会处理                       |

当 skill 提到某个角色（如"打上 AFK-ready triage 标签"），使用此表对应的标签名。

如需自定义标签名，编辑右侧列即可。
