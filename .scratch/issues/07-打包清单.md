# 07 - 打包清单

## Type

AFK

## Blocked by

- 03-旅行计划CRUD

## What to build

Trip 下的独立打包清单页。物品有名称、分类、负责人、勾选状态。

**API**：packing_items 表的 CRUD + check 切换 + 排序路由。

**前端**：
- 从 Trip 详情页顶部 [打包清单] 按钮进入独立页面 `/trips/:id/packing`
- 物品列表：名称、分类标签、负责人（sd/sg）、勾选框
- 添加物品：底部输入框 + 分类 + 负责人选择
- 勾选：点击 checkbox 切换 checked 状态
- 进度条：顶部显示 "已打包 X / Y"
- 排序：拖拽调整
- 删除：软删除

## Acceptance criteria

- [ ] 从行程规划页顶部 [打] 按钮进入打包清单页
- [ ] 列表显示所有物品：名称、分类、负责人、勾选状态
- [ ] 顶部进度条 "已打包 3 / 10"
- [ ] 添加物品：输入名称、选分类、选负责人
- [ ] 勾选/取消勾选即时生效
- [ ] 拖拽排序
- [ ] 删除后从列表消失（软删除）
- [ ] 返回行程规划页正常
