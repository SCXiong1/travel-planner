# 05 - 活动基础管理

## Type

AFK

## Blocked by

- 04-天管理

## What to build

在某一天下管理活动列表。活动有基本字段（类型、名称、位置、时间），支持拖拽排序。

**API**：activities 表的 CRUD + 排序路由。`GET` 按 `sort_order` 排序。

**前端**：
- 右侧活动时间线：选中某天后，显示该天所有活动，按时间排列
- 活动卡片显示：时间、类型标签（吃/住/行/景点，不同颜色）、名称、位置
- 添加活动：底部 [+加活动] 按钮，弹出编辑弹窗
- 编辑弹窗：类型下拉、名称、位置（可选）、开始/结束时间
- 拖拽排序活动列表
- 删除活动：确认弹窗（软删除）

活动类型颜色：吃=橙色、住=蓝色、行=绿色、景点=紫色。

## Acceptance criteria

- [ ] 选中天，右侧显示该天活动时间线
- [ ] 无活动时显示"添加第一个活动"引导
- [ ] 添加活动：弹窗填类型+名称+时间，卡片出现在时间线中
- [ ] 活动卡片显示：时间、彩色类型标签、名称、位置
- [ ] 四种类型用四种颜色区分
- [ ] 编辑活动：点击卡片弹出编辑弹窗
- [ ] 拖拽排序后顺序更新
- [ ] 删除活动后从列表消失（软删除）
- [ ] 切换到不同天，右侧刷新为该天的活动
