# 01 - 项目骨架：数据库 + 应用框架 + Docker

## Type

AFK

## Blocked by

None - can start immediately.

## What to build

搭建项目的最小骨架，让开发环境可跑、Docker 能构建。

**数据库**：SQLite 3，WAL 模式。建所有表（trips, days, activities, packing_items），每张表含 `deleted_at` 字段。写好建表 DDL 和基本的增删改查辅助函数。

**后端**：FastAPI 应用入口，挂载 CORS 中间件，路由占位，启动 uvicorn。提供一个 `/api/health` 端点验证服务正常。

**前端**：Vue 3 + Vite + Tailwind 脚手架。Vite 开发代理配置（`/api` 和 `/ws` 转发到后端）。

**Docker**：多阶段 Dockerfile（构建前端 → 打包进容器），FastAPI serve 静态文件 + API。docker-compose.yml 挂载 data/ 目录。

## Acceptance criteria

- [ ] `cd server && pip install -r requirements.txt && uvicorn main:app --reload` 启动后端
- [ ] `cd client && npm install && npm run dev` 启动前端
- [ ] 前端开发服务器代理 `/api` 到后端
- [ ] `/api/health` 返回 200
- [ ] `docker build -t travel-planner . && docker run -p 3000:3000 travel-planner` 成功
- [ ] 数据库文件在 `data/travel.db`，WAL 模式
- [ ] 所有表已建好（trips, days, activities, packing_items），含 deleted_at
