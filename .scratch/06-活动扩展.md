# 06 - 活动扩展：开销 + 评价 + 预约

## Type

AFK

## Blocked by

- 05-活动基础管理

## What to build

在活动编辑弹窗中增加三个扩展区域，让活动卡片展示更丰富的信息。

**预约**：是否需要预约（开关），开启后显示预约详情输入框（文字，如"已打电话留座，确认号1234"）。

**开销**：金额（数字）、支付人（sd/sg 下拉）、分摊方式（平分/归集 下拉）。三个字段均为可选——不是每个活动都有开销。

**评价**：文字输入框，旅行中或旅行后填写。

**活动卡片**：卡片上显示开销摘要（如 "¥20 sd付 平分"）和评价预览（截断前 50 字）。预约标记为小图标（如📋）。

## Acceptance criteria

- [ ] 编辑弹窗中有"预约"区域：开关 + 详情输入框
- [ ] 需要预约时，活动卡片显示预约图标
- [ ] 编辑弹窗中有"开销"区域：金额、支付人、分摊方式
- [ ] 开销三个字段均可留空
- [ ] 有开销时，卡片显示 "¥XX sd/sg付 平分/归集"
- [ ] 编辑弹窗中有"评价"区域：文字输入框
- [ ] 有评价时，卡片显示评价预览（截断50字）
- [ ] 保存后卡片即时更新
